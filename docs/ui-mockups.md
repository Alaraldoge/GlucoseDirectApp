# UI Mockups & Design Specifications

**Version**: 1.0  
**Last Updated**: 2025-09-30

Visual specifications and mockups for EatThisDie's DOS amber CGA aesthetic.

---

## Design Overview

**Visual Identity**: DOS amber CGA terminal aesthetic with modern iOS usability  
**Primary Color**: `#ffbf00` (Amber)  
**Background**: `#0a0a0a` (Near black)  
**Typography**: Monospace (SF Mono)  
**Corner Radius**: Sharp (0px) or minimal (2px)

---

## Key Screens

### 1. Main Dashboard
- Current glucose reading with vector-style line graph
- Recent meals list with carb totals
- Daily summary statistics
- All text in amber monospace
- Sharp-cornered cards on dark background

### 2. Food Logging
- Camera interface with amber grid overlay
- Real-time food recognition
- Confidence indicators
- Manual adjustment options
- Barcode scanning mode

### 3. Glucose Trends
- DOS-style line charts (green CGA lines)
- Statistical summaries
- Correlation insights
- Time range selectors

### 4. Settings
- HealthKit integration status
- Theme switcher (Amber/Green/Cyan)
- Privacy controls
- Data export options

---

## Component Specs

### DOS Button
- Amber text on dark background
- 2px amber border, sharp corners
- Pressed state: filled amber
- Glow effect on interaction

### DOS Card
- Dark gray background (#1a1a1a)
- Subtle border (#333333)
- Header with ">" prefix
- Monospace content

### Input Fields
- Black background with amber text
- Blinking amber cursor
- Glow on focus
- Sharp borders
